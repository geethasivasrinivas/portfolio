# Geetha Siva Srinivas Portfolio

This is a simple personal portfolio website for Junior Machine Learning Engineer and Entry-Level Data Analyst roles.

## Files

- `index.html` – main portfolio page
- `style.css` – styling
- `script.js` – simple JavaScript
- `assets/Geetha_Siva_Srinivas_Resume.pdf` – resume file

## How to Run Locally

1. Extract the ZIP file.
2. Open `index.html` in your browser.

## How to Deploy on GitHub Pages

1. Create a new GitHub repository named `portfolio`.
2. Upload all files from this folder.
3. Go to repository **Settings**.
4. Open **Pages**.
5. Select branch: `main`.
6. Select folder: `/root`.
7. Click **Save**.

Your portfolio will be live at:

`https://yourusername.github.io/portfolio/`

## What to Edit

Replace these placeholders in `index.html`:

- LinkedIn link
- GitHub link
- Remaining publication titles
- Project GitHub/demo links if available
